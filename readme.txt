LocalGrid Plugin is the best plugin for the make your site as a directory listing ,deals and reviews site .

== Description ==

LocalGrid Plugin is use CityGrid API which has free 18M business list for the USA . User need to get the citygrid publisher key 
to use this plugin thats it .

Important Links:

* <a href="http://developer.citygridmedia.com/dashboard/registration" title="Registration page">CityGrid API key</a>

= Features =

* Business List : You can add custom business list for places,deals,reviews.
* Business list preview : Before add your business list you can preview the result in the admin panel 
* Output Customization : You can customize your Title,arredss,description font color size etc . 
* Sortcode for Business list : you will select the business list from dropdown and set the <code>[localgrid]</code> shortcode in the content .
* Shortcode for the Allcity : To browse all city with all populare category you can just set the <code>[allcity]</code> shortcode in your page .
* Shortcode for Custom Search : To make a custom search for business list you use <code>[localgrid_zipcode]</code> shortcode

== Installation ==

1. 	Install & Activate the plugin

2.	Add a business list 

3. 	Go to your post/page an enter the tag '[localgrid]' after select the business list from the dropdown.

That's it ... Have fun
